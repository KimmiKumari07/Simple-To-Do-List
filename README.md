# Personal To-Do List Application

## Overview
This is a command-line based Personal To-Do List Application developed in Python.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Categorize tasks
- Save tasks using JSON file
- Load tasks automatically

## Requirements
- Python 3.x

## How to Run

1. Open terminal.
2. Navigate to project folder.

Example:

todo.py

## Menu Options

1. Add Task
2. View Tasks
3. Mark Task Completed
4. Delete Task
5. Exit

## Data Storage

Tasks are stored in:

tasks.json

This ensures tasks remain available even after closing the application.
